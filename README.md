# Chatbot Bob

A Java-based chatbot application with multiple interactive features and mini-apps.

## Features

- **Chat Context**: Maintain conversation history
- **Games**: Play built-in games
- **Jokes**: Get jokes from the chatbot
- **Small Talk**: Casual conversation support
- **Weather**: Weather information
- **Time**: Time and date display
- **To-Do List**: Task management
- **Timer**: Timer functionality
- **System Info**: System information display
- **Sound**: Audio feedback

## Building & Running

1. Compile the project in Eclipse or with `javac`
2. Run `src/main/Main.java` or `src/main/MainWindowLauncher.java`

## Project Structure

- `src/main/` - Core application and launcher
- `src/apps/` - Main application modules
- `src/seperateApps/` - Additional standalone apps
